# Publication Design Requirements

## Cover Page
- UNEP logo: top-right corner, minimum 20.0mm width, target 27.5mm, height 30.0mm
- UNEP logo on back cover: bottom-left corner, minimum 20.0mm width
- Main title: 28.0-34.0pt
- Subtitle: 12.0-14.0pt
- Heading color: UNEP Cyan (#00AEEF)
- Partner logos: 10.0mm spacing from other elements

## Typography
- Body text: 9.0-12.0pt, Roboto Flex preferred (Roboto acceptable)
- Chapter titles (H1): 18.0-24.0pt, Roboto Flex preferred (Roboto acceptable), bold, color #00AEEF
- Section headings (H2): 14.0-18.0pt, Roboto Flex preferred (Roboto acceptable), bold, color #00AEEF
- Subsection headings (H3): 12.0-14.0pt, Roboto Flex preferred (Roboto acceptable), color #00AEEF, underlined
- Sub-subsection (H4): 10.0-12.0pt, Roboto Flex preferred (Roboto acceptable), bold
- Captions: 7.0pt, Roboto
- Charts/axis labels: 8.0pt, Roboto Condensed preferred (Roboto acceptable)

## Images
- Minimum DPI: 300 for print quality
- Maximum figure width: 160.0mm
- Color space: RGB or CMYK
- Chart stroke weight: 0.3-0.4pt

## Margins
- Top margin: 20.0-25.0mm
- Bottom margin: 20.0-30.0mm
- Inside margin (binding): 20.0-30.0mm
- Outside margin: 20.0-25.0mm
- Full-bleed images are acceptable and intentional

## Required Elements
- ISBN: Must be present (format: ISBN 978-xxx or 979-xxx)
- DOI: Must be present
- Job number: Must be present - UNEP job number
- Copyright notice: Must be present (format: (C) [year] United Nations Environment Programme)
- Territorial disclaimer: Must be present - designations employed...do not imply...opinion...United Nations
- Commercial products disclaimer: Must be present - mention of...commercial products...does not imply endorsement
- Views expressed disclaimer: Must be present - views expressed...do not necessarily reflect...United Nations
- Suggested citation: Must be present
- SDG icons: Must be present - 1-3 SDG icons should be displayed
- Table of contents: Must be present - tagged and linked
- Page numbers: Must be present - Roman numerals for front matter, Arabic for main content

## Additional Rules
- Publications are formal UNEP documents requiring all metadata
- Pay attention to heading hierarchy consistency
- Cross-reference table of contents with actual page numbers
