# Factsheet Design Requirements

## Cover Page
- UNEP logo: top-right corner, minimum 20mm width, target 27.5mm
- Main heading: 28-34pt
- Subheading: 18-24pt
- Heading color: UNEP Cyan (#00AEEF)

## Typography
- Body text: 11-12pt, Roboto Flex preferred (Roboto acceptable)
- Font weights: Regular or Light
- Captions: 9-10pt
- Chart axis labels: Roboto Condensed, 8pt

## Images
- Minimum DPI: 300 for print quality
- Color space: RGB or CMYK
- All images should have adequate contrast and be clearly visible

## Margins
- Left/right margins: 20-25mm
- Top/bottom margins: 15-20mm
- Full-bleed images are acceptable and intentional

## Required Elements
- Page numbers should be present
- Partner logos (if present) should have 10mm spacing

## Notes
- Factsheets are typically single page or spreads
- Compact layout with emphasis on visual communication
- No ISBN required for factsheets
